# Clojure Tic Tac Toe

## Setup

To play, install [Leiningen](https://leiningen.org/).

`git clone git@github.com:saralein/tic-tac-toe_clojure.git`
`cd tic-tac-toe_clojure`

Then run `lein trampoline run`.

## Test

To test, run `lein test`.
